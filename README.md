# StackSpend

StackSpend is an AI tooling spend audit platform that helps teams identify unnecessary SaaS costs, optimize subscriptions, and uncover savings opportunities.

Built with Next.js, TypeScript, Supabase, Resend, Tailwind CSS, and shadcn/ui.

---

# Features

- AI tooling spend audit
- Dynamic savings recommendations
- Responsive SaaS UI
- Lead capture flow
- Email
